The Virtual Geophysics Laboratory (VGL) is a scientific workflow portal that provides Geophysicists with access to an integrated environment that exploits eResearch tools and Cloud computing technology. It provides in one place -

  • Online access to a wide variety of geophysical datasets
  • Access to specialised geophysics analysis programs
  • Access to high-performance computing
  • A workflow with spatial environment

 



ACKNOWLEDGEMENTS

VGL is a collaboration between AuScope, CSIRO, Geoscience Australia (GA), and the National Computational Infrastructure (NCI).
It has been funded by the Federal Government's Education Investment Funds through Australian Research Data Commons (ARDC).
It was developed in close collaboration with the geophysics user community and representatives from GA, CSIRO, ANU, Monash, and UQ.
